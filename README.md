# AngelHub

> A skill registry for angelic AI agent workflows.

AngelHub provides reusable skills for AI agents across multiple ecosystems.

## Ecosystems

| Ecosystem | Description |
|-----------|-------------|
| [picoclaw](picoclaw/) | PicoCLAW agent skills |

## Quick Start

```bash
# Install a skill
picoclaw skill install keithy/angelhub/picoclaw/skills/self/self-config
```

## Directory Structure

```
angelhub/
└── picoclaw/
    └── skills/
        └── <category>/
            └── <skill_name>/
                └── SKILL.md
```
